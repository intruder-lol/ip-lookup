## :toolbox: Getting Started

### :running: Run Locally

Clone the project

```bash
https://github.com/knok-gg/ip-lookup
```
Go to the project directory
```bash
cd ip lookup.py
```
Install dependencies
```bash
pip install requests fade colorama
```
